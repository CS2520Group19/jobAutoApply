Linked in Application Bot Instructions

***Must setup linkedin account first by uploading a resume***
https://www.linkedin.com/jobs/application-settings/?hideTitle=true

1. Install Selenium
in CMD py -m pip install Selenium

2. Install Microsoft Edge Webdriver
https://developer.microsoft.com/en-us/microsoft-edge/tools/webdriver/
Go to release 109 and download the right webdriver for your operating system

3. Run Program

Program Options Menu
1. Start job search
2. Add Experience. Creates {key:value}
Ex. {java:2}    Answers questions containing the word 'java' and inputs 2 years of experience
3. Change Job Query - changes type of jobs that are being searched for
4. Change Phone number - changes phone number sent to application
